# Clinical Pathways

Clinical pathways is a website where users can get information on medical procedures and conditions. 
Each condition is represented as an interactive flowchart. Users can interact with these procedures by clicking 
on the nodes to reveal more information on that particular section of the medical procedure

## Getting Started

These instructions will help you get you a copy of the project up and running on your local machine for the purpose of developing and testing the project.
### Prerequisites

A working internet connection is required to load some of the external scripts used (Angular, Bootstrap, etc). 

```
For the best experience use a browser on a laptop or desktop
```

```
https://github.com/mchimbakate/mchimbakate.github.io
```

Step Two: Extract the files to your prefered location on your machine

Step Three: Open index.html using a browser of your choice


## How to use the site

Use the search bar to search for a condition
```
*If the link is active, then click on it,
*If the link is not active, the pathway for that condition/procedure has not yet been implemented.
```

After clicking on one of the active links, you will be presented with a flowchart.
To view more information about a particular section of the flowchart, 
simply click on that section and a modal will appear with more information.

To exit, click 'x' or the 'close' button or simply click anywhere on the page.
